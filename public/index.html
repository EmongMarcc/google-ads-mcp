Token expired?

⚠️

Getting an invalid_grant error?

OAuth refresh tokens expire after 6 months of inactivity or if your Google password changes. When this happens, all MCP tools will return a clear error message explaining what to do.

To get a new refresh token: Visit the Google OAuth Playground, select the Google Ads API scope (https://www.googleapis.com/auth/adwords), and exchange for a new token. Then re-register above with your new refresh token.
